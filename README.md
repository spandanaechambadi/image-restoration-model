# Image Restoration with Failure Analysis

## 📌 Problem
Improve image restoration performance and analyze failure cases in generative models.

## ⚙️ Tech Stack
- Python
- PyTorch
- Deep Learning

## 🚀 Approach
- Developed a multi-task restoration model handling multiple degradation types
- Compared performance with ESRGAN
- Conducted systematic failure analysis to understand error propagation

## 📊 Results
- Achieved 26.92 dB PSNR across 5 degradation types on 30K images

## 💡 Impact
Provides insights into limitations of generative restoration models and improves robustness.
